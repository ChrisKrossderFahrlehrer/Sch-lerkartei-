# Schülerkartei – Fahrschul-Verwaltung

PWA für Fahrlehrer zur digitalen Verwaltung von Fahrschülern.

## Dateien

| Datei | Beschreibung |
|---|---|
| `index.html` | Die komplette App (eine einzige Datei) |
| `sw.js` | Service Worker für Offline-Modus |

## Update einspielen

1. Neue `index.html` herunterladen
2. Auf GitHub → Repository öffnen
3. Auf `index.html` klicken → Stift-Symbol (Edit) → „Upload file" oder direkt ersetzen
4. „Commit changes" klicken
5. Fertig – App ist nach ~1 Minute aktuell

## Firebase Projekt

- **Projekt:** fahrschule-ebc65
- **Region:** europe-west3 (Frankfurt)
- **Auth:** Benutzername + Passwort

## Nach URL-Änderung

In der Firebase Console → Authentication → Authorized Domains → neue GitHub-URL eintragen:
`DEINNAME.github.io`
